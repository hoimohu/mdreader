# なんだこれ

自分用のmd変換  
なんとなくで作ったから信用してはいけない

# たぶん使えるやつ

* pre code
* code
* blockquote
* hr
* ul|ol (おそらく一部機能に不備が...)
* table
* em|strong|strong em|code|del|sup|sub|mark|(脚注も一応できる)
* br
* a
* img
* h1~6
* p
* checkbox

# 絶対できないこと

* emoji
* blockquoteにスペースを空けてからblockquoteを入れようとすること

# 注意

* 適当に作ってあります。おそらく不具合だらけ
* urlの自動リンク化はしません
